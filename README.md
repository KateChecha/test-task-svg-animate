# SVG & Chart Demo

Демонстрационное приложение на Vue 3: круговой прогресс-бар (SVG) и интерактивная круговая диаграмма (Chart.js).

## Возможности

- **Прогресс-бар** — SVG-компонент с настройкой процента, статуса, толщины, размера. Режимы: круг и панель. Автоматическая анимация.
- **Круговая диаграмма** — добавление, редактирование и удаление секторов. Выбор цвета через color picker.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173).

## Сборка

```bash
npm run build
```

## Стек

Vue 3, TypeScript, Vite, Vue Router, Chart.js, vue-chartjs, vue3-colorpicker, Sass
